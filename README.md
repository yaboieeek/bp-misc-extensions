# BP Misc Extensions 
Userscripts for backpack.tf that are going to be placed in one script eventually

## $BP Api Lookup 
<p>Script adds a switch for the default search bar that toggles the mode. If it's disabled it acts like a normal search. If it's enabled, it looks up the name via snapshots api (loads listings and shows top)</p>

## $BP Time Normalizer 
<p>Script converts all the found timestamps on page to UTC time keeping the 'x ago'. If it's a suggestion timestamp, it will highlight 3 months old suggestions timestamps with red color</p>
